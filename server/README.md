# cosmic-pairing api

## Project setup

### Install Flask 
```
pip install Flask==1.0.2 Flask-Cors==3.0.7
```

### Install virtualenv (venv) 
```
python3 -m venv
```

### Create venv
```
virtualenv venv
```

---

## Workflow commands

### Activate venv
```
source venv/bin/activate
```

### Deactivate venv
```
deactivate
```

### Start app
For now, manually export `FLASK_APP` and `FLASK_ENV` variables before running:
```
flask run
```
